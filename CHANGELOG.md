# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

## [0.1.0]
### Added
- Инициализация репозитория: `.gitignore`, `README.md`, `requirements.txt`
- Базовая структура папок `src/`, `tests/`
- Модуль `camera.py` с классом `CameraCapture` для работы с камерой
- Модуль `detection.py` с функцией-заглушкой `detect_faces`
- Точка входа `main.py` с простым циклом захвата видео

## [0.2.0]
### Added
- Модуль `emotion_analyzer.py` с функцией `analyze_emotions` на базе FER (с увеличением области лица)
### Changed
- Модуль `detection.py`: функция `detect_faces` теперь масштабирует кадр для ускорения и возвращает координаты в оригинальном размере
- `main.py`: теперь использует детекцию и анализ на одном тестовом кадре
